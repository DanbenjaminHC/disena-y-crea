# Archivos Públicos

Esta carpeta contiene archivos públicos que pueden ser compartidos y accedidos por cualquier usuario.

## Descripción
Aquí puedes subir y organizar archivos que desees que sean de acceso público.

## Estructura
- Organiza los archivos en subcarpetas según el tipo o proyecto
- Asegúrate de que los archivos no contengan información sensible

## Uso
Para agregar nuevos archivos a esta carpeta, simplemente subirlos aquí manteniendo una estructura clara y ordenada.
